Detailed Description

Vector of Mastodon::param, used for passing parameters in calls.

The only difference to a std::vector<param> is the added member find.

Example:

parameters p =
{
{ "media_ids", { "1234", "4321" } },
{ "status", { "Hello world!" } }
};
